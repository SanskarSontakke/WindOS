# WindOS User Guide

## Getting Started

1.  **Booting**: Wait for the system to initialize.
2.  **Login**: Enter the PIN on the Lock Screen. Default PIN is `1234`.

## The Desktop

-   **Start Menu**: Click the Blue/Teal "Start" button in the bottom-left to open the menu.
-   **Taskbar**: Shows running applications. Active application is highlighted.
-   **Wallpaper**: Click the very top-right corner of the screen to cycle through wallpapers.

## Applications

### Settings
Customize your experience.
-   **Personalization**: Change Theme Color (Start button highlight) and Taskbar Color.
-   **Security**: Change your Login PIN.
-   **Save**: Click the Green "Save" button to apply changes permanently.

### File Explorer
Manage your files.
-   **Navigate**: Up/Down arrows to select, Enter to open.
-   **Go Back**: Backspace.
-   **New Folder**: Press `F2` and type a name.
-   **Delete**: Press `Delete` to remove a selected file or empty folder.
-   **Open**: Text files (`.txt`, `.wnd`) open in Notepad.

### Notepad
Simple text editor.
-   **Save**: Press `F1`. If it's a new file, type the name and press Enter.
-   **Exit**: Switch apps via the Start Menu.

### Clock
-   **Clock**: Shows current time.
-   **Stopwatch**: Start/Stop/Reset timer.
-   **Timer**: Enter seconds and press Enter to start.

### Calculator
Basic GUI calculator. Supports `+`, `-`, `*`, `/`.

### Console
Command-line interface.
-   `help`: List commands.
-   `cls`: Clear screen.
-   `reboot` / `shutdown`: Power options.

## Troubleshooting

-   **System Freeze**: If an app crashes, the system handles exceptions, but a hard freeze requires a reboot.
-   **File Errors**: Ensure you have write permissions (standard in Cosmos VFS).
